## IPL Data Analysis Dashboard (2008–2025)
This project is an end-to-end data analytics dashboard focused on the Indian Premier League (IPL), covering all seasons from 2008 to 2025. The goal of this project is to transform raw cricket data into meaningful insights using data visualization and business intelligence techniques.

The dashboard provides a complete overview of IPL performance at both team and player levels. It highlights key metrics such as total matches played, number of teams, venues, boundaries (4s and 6s), centuries, and half-centuries. Along with this, it also showcases season-specific insights like the winning team and runner-up, giving a quick summary of each IPL season.

One of the main features of this dashboard is its ability to dynamically analyze data based on the selected season. Using filters, users can explore different years and instantly view updated statistics, making the dashboard interactive and user-friendly.

On the player performance side, I have included detailed insights such as:

* Orange Cap holder (highest run scorer) with total runs and team details
* Purple Cap holder (highest wicket taker) with wicket count and team
* Top performers in terms of most 4s and most 6s

Additionally, the dashboard includes a points table that provides a structured view of team performance in a selected season. It displays matches played, wins, losses, ties, and total points, helping users easily compare team standings.

The main objective of this project is to demonstrate how raw sports data can be converted into actionable insights using modern BI tools. This dashboard helps users quickly understand trends, compare performances, and explore IPL data in an interactive way.
